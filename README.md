# Resume AI Platform 🚀

A high-performance, AI-driven resume optimization platform built with the Gemini 2.0 engine. Land more interviews with data-backed precision, ATS-friendly formatting, and strategic keyword optimization.

![Project Preview](https://via.placeholder.com/800x400?text=Resume+AI+Platform+Preview)

## ✨ Features

- **Neural Analysis**: Instant feedback powered by Google Gemini 2.0.
- **ATS Optimization**: Match your resume to specific job descriptions with high precision.
- **Interactive Dashboard**: Track your optimization history and improvements.
- **Premium Export**: Professional PDF and Text exports for your optimized resumes.
- **Secure Processing**: End-to-end encrypted analysis and secure Google Authentication.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, Framer Motion, Lucide React, Tailwind CSS.
- **Backend**: Node.js, Express, Multer.
- **Database & Auth**: Supabase.
- **AI Engine**: Google Gemini 2.0 API.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn
- Supabase account
- Google Gemini API Key

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/resume-ai.git
   cd resume-ai
   ```

2. **Setup Backend**:
   ```bash
   cd server
   npm install
   # Create a .env file with your credentials (see .env.example)
   npm run dev
   ```

3. **Setup Frontend**:
   ```bash
   cd ../client
   npm install
   # Create a .env file with your credentials (see .env.example)
   npm run dev
   ```
